## Implemented CUDA Kernels

- **KernelAdd** – element-wise addition of two vectors
- **KernelMul** – element-wise multiplication of two vectors
- **KernelMatrixAdd** – addition of two matrices using pitched 2D memory
- **MatrixVectorMul** – matrix–vector multiplication
- **ScalarMul** – dot product of two vectors using parallel reduction
- **CosineVector** – cosine of the angle between two vectors
- **MatrixMul** – matrix–matrix multiplication using shared memory tiling
- **Filter** – filtering array elements based on comparison conditions
